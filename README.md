# KSUWeatherApp


## 概要

「KSU」は九州産業大学が作成して，九州産業大学が提供する公式のスマートフォンアプリ（iPadも対応！）です。

今回は実装した天気機能を公開！

iOS16で新たに登場したWeatherKitを使用

## 開発環境

- Xcode Version 14.2 or later
- iOS 16.2 or later 
- Swift 5.7
- SwiftUI 4.0
